The Complete Guide to Computer Networks

From LANs to the OSI Model, explore the backbone of the digital world.

1. Introduction to Computer Networks

Definition:

A computer network is a set of computers and devices connected to share resources (files, printers, internet) and exchange data.

Main Goals:

  • Resource Sharing: printers, files, apps
  • Communication: email, video conferencing
  • Scalability: adding new systems easily
  • Reliability: backup, multiple paths

Types of Networks:

  • LAN: Small area (office, building). High speed.
  • MAN: Covers a city or large campus.
  • WAN: Covers countries/continents (e.g., Internet).
  • PAN: Personal devices (Bluetooth, hotspot).

Topologies:

  • Bus: single backbone, cheap, but one failure = network crash.
  • Star: central hub/switch, easy to manage, single point failure.
  • Ring: token passing, predictable, but failure breaks network.
  • Mesh: every node connected to every other, fault-tolerant, expensive.
  • Tree/Hybrid: mix of topologies.

2. Network Models

OSI Model (7 Layers)

Each layer has a responsibility:

  • Physical: transmission of raw bits (cables, hubs).
  • Data Link: error detection/correction, framing (switches).
  • Network: routing, logical addressing (IP, routers).
  • Transport: reliability, error recovery (TCP, UDP).
  • Session: manages sessions (start/stop communication).
  • Presentation: data formatting, encryption, compression.
  • Application: end-user services (HTTP, FTP, DNS).

3. Physical Layer

Responsible for transmission of raw bits over physical medium.

Mediums:

  • Wired: Coaxial, Twisted Pair, Fiber Optic.
  • Wireless: Radio, Microwave, Infrared.

Techniques:

  • Multiplexing: FDM (Frequency Division), TDM (Time Division), WDM (Wavelength Division)
  • Switching Techniques: Circuit Switching (telephone, dedicated path), Packet Switching (internet, more efficient), Message Switching (store-and-forward, old system)

4. Data Link Layer

Ensures reliable transmission between directly connected nodes.

Functions:

Framing, Error Detection/Correction, Flow Control, MAC.

Error Detection & Correction:

  • Parity bit, Checksum, CRC (Cyclic Redundancy Check), Hamming Code (can detect & correct errors)

Flow Control:

  • Stop-and-Wait ARQ: send one frame at a time, wait for ACK.
  • Sliding Window Protocol: Go-Back-N (retransmit from error onwards), Selective Repeat (only retransmit lost packets).

MAC Protocols:

  • ALOHA, Slotted ALOHA
  • CSMA (Carrier Sense Multiple Access)
  • CSMA/CD â†’ Ethernet (collision detection).
  • CSMA/CA â†’ WiFi (collision avoidance).
