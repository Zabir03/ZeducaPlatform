Basic Structure of a Java Program

public class Main {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
  • public class Main: Every application must have at least one class definition. The class name should match the filename (Main.java).
  • public static void main(String[] args): This is the main method. It's public, static (belongs to the class, not an object), returns nothing (void), and accepts an array of strings as arguments.
  • System.out.println(): A built-in method to print text to the console.

Variables & Data Types

Java has two categories of data types: primitive types and reference types.

Primitive Data Types

  • Numeric Types: byte, short, int, long (for integers); float, double (for floating-point numbers).
  • Character Type: char for single characters.
  • Boolean Type: boolean for true or false values.

Reference Data Types

These refer to objects. Examples include String, Arrays, and any Class you create. They hold a memory address of an object rather than the object itself.

Operators

Java supports a rich set of operators to perform operations on variables.

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, > ,<,>=, <=
  • Logical: && (AND), || (OR), ! (NOT)
  • Assignment: =, +=, -=

Control Flow

Control the flow of your program using conditional statements and loops.

Conditionals and Loops

  • Conditionals: if-else, if-else-if, switch
  • Loops: for, while, do-while, and the enhanced for-each loop for iterating over collections.

OOP Concepts

Java is built on the principles of Object-Oriented Programming, which help create modular, flexible, and reusable code.

  • Encapsulation: Hiding data within a class.
  • Abstraction: Hiding implementation details.
  • Inheritance: Reusing code from other classes.
  • Polymorphism: Allowing objects to take many forms.

Classes & Objects

A class is a blueprint from which objects are created. An object is an instance of a class, with its own state (fields) and behavior (methods).

Constructors

A constructor is a special method that is used to initialize objects. It is called when an object of a class is created.

Inheritance

Inheritance allows one class (subclass) to acquire the properties and methods of another class (superclass). This is achieved using the extends keyword.

Video Explanation

Polymorphism

Polymorphism means "many forms". In Java, it allows us to perform a single action in different ways. It is achieved through method overloading (compile-time) and method overriding (run-time).

Video Explanation

Abstraction

Abstraction is the process of hiding the implementation details and showing only functionality to the user. It can be achieved with abstract classes and interfaces.

Video Explanation

Encapsulation

Encapsulation is the wrapping of data (variables) and code acting on the data (methods) together as a single unit. It's achieved by declaring variables as private and providing public getter and setter methods.

Video Explanation

Exception Handling

Java's exception handling is a powerful mechanism to handle runtime errors. It uses try, catch, finally, throw, and throws keywords.

Video Explanation

Collections Framework

The Java Collections Framework provides an architecture to store and manipulate groups of objects. It includes interfaces like List, Set, Map and implementations like ArrayList, HashSet, and HashMap.
