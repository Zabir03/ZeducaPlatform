1. Introduction to C++

History & Purpose

Developed by Bjarne Stroustrup at Bell Labs starting in 1979, C++ was created as an extension of the C language. Its primary goal was to add object-oriented features to C, allowing for better organization and management of large-scale software projects.

Key Features

  • Hybrid Language: Combines low-level capabilities (like C) with high-level object-oriented features.
  • Performance: As a compiled language, C++ is known for its speed and efficiency, making it a top choice for game engines, high-frequency trading, and system software.
  • Object-Oriented: Fully supports OOP principles like encapsulation, inheritance, and polymorphism.
  • STL: The Standard Template Library provides a rich set of data structures and algorithms.

2. Basic Structure of a C++ Program

Every C++ program has a fundamental structure that includes necessary headers, a namespace declaration, and a `main` function where execution begins.

#include 

// This line allows us to use names for objects and variables from the standard library.
using namespace std;

// The main function is the entry point of every C++ program
int main() {
    // cout is used for output. << is the insertion operator.
    cout << "Hello, World!"; 
    
    // return 0 indicates that the program ended successfully.
    return 0;
}

3. Data Types and Variables

Variables are named storage locations. In C++, every variable has a data type that determines the kind of data it can hold.

Categories of Data Types

  • Primitive Types: int, float, double, char, bool. These are the fundamental building blocks.
  • Derived Types: array, pointer, reference. These are built from primitive types.
  • User-Defined Types: class, struct, enum. These are custom types you can create.

4. Operators

Operators are symbols that perform operations. C++ has a wide range of operators.

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: && (AND), || (OR), ! (NOT)
  • Bitwise: &, |, ^, ~, <<, >>

5. Control Statements

Control statements alter the flow of execution in a program.

Decision Making & Loops

  • Decision Making: if-else, switch-case
  • Loops: for, while, do-while for repetitive tasks.
  • Jump Statements: break, continue, goto to alter the loop/switch flow.

9. Object-Oriented Programming (OOP)

This is the core paradigm of C++. It helps in building modular and reusable code.

(a) Classes and Objects

A class is a blueprint for creating objects (instances).

(b) Encapsulation

Binding data and the functions that manipulate them together. Access is controlled by public, protected, and private specifiers.

(c) Inheritance

A class can inherit properties from another class, promoting code reuse.

(d) Polymorphism

"Many forms." Achieved via function/operator overloading (compile-time) and virtual functions (run-time).

(e) Abstraction

Hiding complex implementation details and showing only the necessary features.

13. Standard Template Library (STL)

A set of C++ template classes to provide common data structures and functions.

  • Containers: vector, list, map, set, etc.
  • Algorithms: sort(), find(), binary_search(), etc.
  • Iterators: Objects that act like pointers to access container elements.

14. Advanced C++

Modern C++ (C++11 and later) introduced powerful features like smart pointers (unique_ptr, shared_ptr) for automatic memory management, lambda expressions for anonymous functions, and multithreading support.
