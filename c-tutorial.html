Introduction to C

C is a powerful, general-purpose programming language developed in the early 1970s by Dennis Ritchie at Bell Labs. It is a procedural language, which means that programs are written as a series of steps or procedures.

Why Learn C?

  • Foundation of Modern Programming: Many modern languages like C++, C#, Java, and Python are influenced by C's syntax and concepts.
  • Performance: C provides low-level memory access and is very fast, making it ideal for system programming, embedded systems, and game development.
  • Portability: C programs can be compiled on a wide variety of computer platforms.

Your First C Program: "Hello, World!"

#include <stdio.h>

int main() {
    // This line prints the text to the console
    printf("Hello, World!\\n");
    return 0;
}

Variables & Data Types

Variables are containers for storing data values. In C, each variable must have a specific data type, which determines the size and layout of the variable's memory.

Basic Data Types

  • int: Stores integers (whole numbers), such as 123 or -456.
  • float: Stores single-precision floating-point numbers, with decimals, such as 19.99.
  • double: Stores double-precision floating-point numbers, for greater precision.
  • char: Stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes.

Declaring Variables

#include <stdio.h>

int main() {
    int myNum = 15;            // Integer
    float myFloatNum = 5.99;   // Floating point number
    char myLetter = 'D';       // Character

    printf("My number is %d\n", myNum);
    printf("My float is %f\n", myFloatNum);
    printf("My letter is %c\n", myLetter);

    return 0;
}

Operators in C

Operators are symbols that perform operations on variables and values. C supports a rich set of built-in operators.

Types of Operators

  • Arithmetic Operators: +, -, *, /, % (modulus)
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: && (AND), || (OR), ! (NOT)
  • Assignment Operators: =, +=, -=, *=, etc.
  • Increment/Decrement Operators: ++, --

Control Flow (Conditional Statements)

Control flow statements allow you to execute certain blocks of code based on specified conditions. The primary way to control program flow is with if...else and switch statements.

if...else if...else Statement

int number = 0;

if (number > 0) {
    printf("The number is positive.");
} else if (number < 0) {
    printf("The number is negative.");
} else {
    printf("The number is 0.");
}
// Outputs "The number is 0."

Loops

Loops can execute a block of code as long as a specified condition is reached. They are essential for repeating tasks without rewriting code.

for Loop

Used when you know how many times you want to loop.

for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}

while Loop

Loops as long as a condition is true. The condition is checked *before* the loop body is executed.

int i = 0;
while (i < 5) {
    printf("%d\n", i);
    i++;
}

Functions

A function is a block of code that performs a specific task. Functions allow you to break down large programs into smaller, manageable, and reusable pieces.

Defining and Calling a Function

A function has a return type, a name, parameters, and a body. The main() function is special; it's the entry point of every C program.

#include <stdio.h>

// Function declaration (prototype)
int add(int a, int b);

int main() {
    int result = add(5, 3); // Function call
    printf("Result is: %d\n", result);
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b; // Return value
}

Arrays & Strings

An array is a variable that can store multiple values of the same type. In C, a string is simply an array of characters ending with a special null character (\0).

Arrays

#include <stdio.h>

int main() {
    int myNumbers[] = {25, 50, 75, 100};
    
    // Accessing an element (index starts at 0)
    printf("%d\n", myNumbers[0]); // Outputs 25

    // Changing an element
    myNumbers[0] = 33;
    printf("%d\n", myNumbers[0]); // Outputs 33

    return 0;
}

Strings

#include <stdio.h>

int main() {
    char greeting[] = "Hello World!";
    printf("%s\n", greeting);
    return 0;
}

Pointers

A pointer is one of C's most powerful and unique features. It is a variable that stores the memory address of another variable. This allows for direct memory manipulation and is essential for dynamic memory allocation.

How Pointers Work

  • The & (address-of) operator gives you the memory address of a variable.
  • A pointer is declared with an asterisk *, like int *ptr;.
  • The * (dereference) operator lets you access the value stored at the memory address the pointer is pointing to.
#include <stdio.h>

int main() {
    int myAge = 43;      // An int variable
    int* ptr = &myAge;   // A pointer variable, with the name ptr, that stores the address of myAge

    // Output the value of myAge (43)
    printf("%d\n", myAge);

    // Output the memory address of myAge
    printf("%p\n", &myAge);

    // Output the memory address of myAge with the pointer
    printf("%p\n", ptr);

    // Access the value at the address the pointer is pointing to (43)
    printf("%d\n", *ptr);

    return 0;
}

Structures

A structure (struct) is a user-defined data type that allows you to group different data types into a single unit. It's useful for creating records or complex data representations.

Creating and Using a Structure

You define a structure using the struct keyword. To access the members of a structure variable, you use the dot operator (.).

#include <stdio.h>
#include <string.h>

// Define a structure
struct Person {
    char name[50];
    int age;
    float height;
};

int main() {
    // Create a Person structure variable
    struct Person person1;

    // Assign values to its members
    strcpy(person1.name, "John Doe");
    person1.age = 30;
    person1.height = 5.9;

    // Print the members
    printf("Name: %s\n", person1.name);
    printf("Age: %d\n", person1.age);
    printf("Height: %.1f\n", person1.height);
    
    return 0;
}
